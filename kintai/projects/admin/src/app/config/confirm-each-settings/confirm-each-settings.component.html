各種設定 / 初期設定一覧

初期設定

スタッフ登録

サービスを利用するスタッフを登録できます。各スタッフの入社日や時給・交通費、基本的なシフトなどの確認・設定も可能です。

基本情報設定

所定時間･残業･深夜設定

就業規則に基づいた所定時間/時間外の集計/深夜時間を設定できます。

打刻まるめ設定

出勤/退勤/合計勤務時間を切り捨てる設定を登録できます。

休日設定

会社の公休日/法定休日をカレンダーへ登録することができます。

自動休憩設定

労働時間に対して、自動的に休憩を引く設定が登録できます。

休暇・申請管理

特別休暇名の登録

有休/代休/振休以外に30個まで特別休暇を設定できます。

休暇タイプ設定

スタッフが取得可能な休暇の詳細を設定します。休暇申請時の選択肢として追加されます。

その他

出勤簿一括ダウンロード

出勤簿形式で、勤怠実績の出力フォーマットを作成できます。

勤務データダウンロード

表形式で、給与計算などに用いる出力フォーマットを作成できます。
